# HackMIT 2018 Splash Page
Splash page for HackMIT 2018!

## Development
 1. Install Node
 2. `npm install`
 3. `gulp dev` (if something breaks, use `gulp dev` to build, and then `python -m SimpleHTTPServer` to view)
